# GDAN Timer

An ingame timer/countdown for [GD All Nighter](https://gdallnighter.com/).

## What is GD All Nighter?

GD All Nighter is a Geometry Dash Level Creation event hosted by [Herozer](https://www.youtube.com/@herozer).

## Warning
This mod will be functionally useless at the end of each GDAN volume. Downloading after June 14th, 2026 at 12:00:00 PM EST will render this mod USELESS until GDAN Vol. 4.
